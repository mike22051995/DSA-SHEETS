## 🔥 Critical Patterns You MUST Know

### 1️⃣ **BFS Template (Most Important!)**
```cpp
queue<TreeNode*> q;
q.push(root);

while (!q.empty()) {
    int size = q.size();
    for (int i = 0; i < size; i++) {
        TreeNode* node = q.front();
        q.pop();
        
        // Process node
        
        if (node->left) q.push(node->left);
        if (node->right) q.push(node->right);
    }
}
```

### 2️⃣ **DFS Inorder Template**
```cpp
void inorder(TreeNode* root) {
    if (!root) return;
    inorder(root->left);
    // Process root
    inorder(root->right);
}
```

### 3️⃣ **Validate BST Range Pattern**
```cpp
bool isValid(TreeNode* root, long min, long max) {
    if (!root) return true;
    if (root->val <= min || root->val >= max) return false;
    return isValid(root->left, min, root->val) && 
           isValid(root->right, root->val, max);
}
```

### 4️⃣ **LCA Pattern**
```cpp
TreeNode* LCA(TreeNode* root, TreeNode* p, TreeNode* q) {
    if (!root || root == p || root == q) return root;
    TreeNode* left = LCA(root->left, p, q);
    TreeNode* right = LCA(root->right, p, q);
    if (left && right) return root;
    return left ? left : right;
}
```

## 💡 Interview Tips

### **Before the Interview:**
1. ✅ Sleep well - tree problems require clear thinking
2. ✅ Review BFS and DFS templates
3. ✅ Practice explaining solutions out loud
4. ✅ Have a pen and paper ready for the interview

### **During the Interview:**
1. **Clarify the problem:**
   - Is it a BST or regular binary tree?
   - Can there be duplicate values?
   - What to return for empty tree?

2. **Think out loud:**
   - "This looks like a level order traversal problem"
   - "I can use BFS here because..."
   - "Let me trace through an example"

3. **Start with brute force, optimize later:**
   - "The naive approach would be O(n²)..."
   - "We can optimize using a hashmap to O(n)"

4. **Draw examples:**
   - Always draw a tree
   - Trace your algorithm on it
   - Check edge cases (null, single node, skewed tree)

### **Common Edge Cases:**
- ✅ Empty tree (root = null)
- ✅ Single node tree
- ✅ Skewed tree (all left or all right)
- ✅ Perfect binary tree
- ✅ Complete binary tree

## 🎓 Time & Space Complexity Cheat Sheet

| Operation | Time | Space | Notes |
|-----------|------|-------|-------|
| DFS Traversal | O(n) | O(h) | h = height, worst case O(n) |
| BFS Traversal | O(n) | O(w) | w = max width |
| BST Search | O(h) | O(h) | O(log n) balanced, O(n) skewed |
| Validate BST | O(n) | O(h) | Must check all nodes |
| LCA | O(n) | O(h) | Single pass |
| Serialize | O(n) | O(n) | Store all nodes |
