## 🎯 How to Use

1. **Browse by Topic**: Navigate to specific folders based on the data structure or algorithm you want to practice
2. **Problem Numbering**: Files are numbered for easy sequential learning (e.g., `01_two_sum.cpp`, `02_best_time_to_buy_sell_stock.cpp`)
3. **Study Approach**:
   - Read the problem statement in the code comments
   - Try solving it yourself first
   - Review the solution and understand the approach
   - Analyze time and space complexity

## 💻 Compilation and Execution

To compile and run any C++ file:

```bash
g++ -std=c++17 filename.cpp -o output
./output
```
