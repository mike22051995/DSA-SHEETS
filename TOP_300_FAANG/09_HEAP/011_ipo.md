# IPO

## Description
Choose at most k projects to maximize capital. Each project requires capital[i]
and yields profit[i]. Start with capital w.

## Input Format
- k: maximum number of projects
- w: initial capital
- profits: list of project profits
- capital: list of required capital

## Output Format
- maximum capital after completing up to k projects

## Approach
- Sort projects by required capital.
- Use a max-heap of profits for all projects affordable at current capital.
- Repeatedly pick the most profitable available project.

## Complexity
- Time: O(n log n)
- Space: O(n)

## Sample Input/Output

Sample 1
```
Input:  k = 2, w = 0, profits = [1,2,3], capital = [0,1,1]
Output: 4
```

Sample 2
```
Input:  k = 1, w = 2, profits = [1,2,3], capital = [1,1,2]
Output: 5
```

Sample 3
```
Input:  k = 3, w = 0, profits = [1,2], capital = [1,1]
Output: 0
```

Sample 4
```
Input:  k = 2, w = 1, profits = [2,4,6], capital = [0,1,1]
Output: 11
```

Sample 5
```
Input:  k = 2, w = 3, profits = [1,2,3], capital = [0,4,5]
Output: 6
```
